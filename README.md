## Usage

Each file is self-contained with a table of contents. Open the file for your shell, jump to the relevant category, and find the command with its flags, syntax, and examples.

For quick lookups use your editor's search (`Cmd+F` / `Ctrl+F`) or grep:

```bash
grep -n "curl" linux_terminal_commands_full.md
```
